Masscan

About Masscan

Masscan is a high-speed port scanner that can scan the entire Internet in a matter of minutes. It is similar to Nmap but optimized for speed and efficiency, making it capable of scanning over 10 million packets per second. Masscan is useful for network administrators and security professionals who need to perform large-scale port scans for network inventory, security assessments, and research purposes.

Masscan uses asynchronous transmission to achieve its speed and can perform both TCP SYN scans and banner grabbing. Due to its powerful capabilities, it should be used responsibly and ethically.


Installation

Masscan is available for Linux and can be compiled from source. Below are the installation steps for common platforms.

Linux Installation (Ubuntu/Debian):

  1. Install the required dependencies:
    2. sudo apt update && sudo apt install git gcc make libpcap-dev
  2. Clone the Masscan repository:
    3. git clone https://github.com/robertdavidgraham/masscan
  3. Navigate to the Masscan directory:
    4. cd masscan
  4. Compile Masscan:
    5. make
  5. Optionally, install Masscan system-wide:
    6. sudo make install

macOS Installation:

  1. Install Xcode command-line tools:
    2. xcode-select --install
  2. Install Homebrew if not already installed:
    3. /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  3. Install Masscan using Homebrew:
    4. brew install masscan

Basic Usage of Masscan

Masscan requires root or sudo privileges to send raw packets. Below are some basic commands and examples.

Scanning a Single IP Address

To scan a single IP address on a specific port:

sudo masscan 192.168.1.1 -p80

This scans port 80 on the IP address 192.168.1.1.


Scanning an IP Range

To scan a range of IP addresses on specific ports:

sudo masscan 192.168.1.0/24 -p80,443,22

This scans ports 80, 443, and 22 on the subnet 192.168.1.0/24.


Scanning the Entire Internet

To perform a wide-scale scan (use with caution and proper authorization):

sudo masscan 0.0.0.0/0 -p80 --rate=100000

This scans port 80 on all IPv4 addresses at a rate of 100,000 packets per second.


Adjusting the Scan Rate

To modify the packet transmission rate:

sudo masscan 192.168.1.0/24 -p80 --rate=1000

This sets the rate to 1,000 packets per second.


Advanced Masscan Techniques

Output Options

To save the scan results to a file:

sudo masscan 192.168.1.0/24 -p80 -oX scan_results.xml

This saves the output in XML format.


Excluding IP Addresses

To exclude specific IP addresses or ranges from the scan:

sudo masscan 192.168.1.0/24 -p80 --excludefile exclude.txt

Create an exclude.txt file with IP addresses or ranges to exclude.


Using Custom Source IP and Port

To specify a source IP address and port:

sudo masscan 192.168.1.0/24 -p80 --source-ip 192.168.1.100 --source-port 60000

Banner Grabbing

Masscan can perform banner grabbing to retrieve service information:

sudo masscan 192.168.1.0/24 -p80 --banners

This attempts to grab banners from services running on port 80.


Use Cases of Masscan

  • Network Inventory: Quickly identify active hosts and open ports on large networks.
  • Security Assessments: Detect potential vulnerabilities by identifying services exposed on the network.
  • Research and Analysis: Collect data on Internet-wide services for research purposes.
  • Compliance Auditing: Verify that unauthorized services are not running on the network.
  • Monitoring Changes: Track changes in network services over time.

Best Practices and Tips

  • Use Responsibly: Masscan can generate significant network traffic; use it responsibly and with permission.
  • Avoid Scanning the Entire Internet Without Authorization: Unauthorized scanning can lead to legal issues and network abuse complaints.
  • Adjust Scan Rate Carefully: High scan rates can overwhelm networks and systems; start with lower rates.
  • Whitelist Your IP: If scanning within an organization, ensure your scanning IP is whitelisted in intrusion detection systems.
  • Combine with Nmap: Use Masscan for initial discovery and Nmap for detailed scanning of specific targets.

Common Masscan Options Summary

Option Description
-p <ports> Specify ports to scan (e.g., -p80,443)
--rate <number> Set the packet transmission rate (packets per second)
-oX <file> Output results in XML format to a file
-oG <file> Output results in Grepable format
--excludefile <file> Exclude IP addresses or ranges listed in a file
--source-ip <ip> Set the source IP address for packets
--source-port <port> Set the source port for packets
--banners Enable banner grabbing
--interface <iface> Specify the network interface to use
--ping Ping scan mode

Legal and Ethical Considerations

Important: Masscan can generate large volumes of network traffic and can be disruptive if used improperly. Unauthorized scanning of networks or IP addresses is illegal in many jurisdictions and can lead to serious legal consequences. Always ensure you have explicit permission from the network owner before conducting any scans. Use Masscan responsibly, ethically, and in compliance with all applicable laws and regulations.
