Hashcat

About Hashcat

Hashcat is the world's fastest and most advanced password recovery utility, supporting five unique modes of attack for over 300 highly-optimized hashing algorithms. hashcat currently supports CPUs, GPUs, and other hardware accelerators on Linux, Windows, and macOS, and has facilities to help enable distributed password cracking.

Installation

Linux installation:

sudo apt install hashcat

Basic Syntax (example)

hashcat -m 3200 -T 4 -a 0 -o CRACKED.txt HASH.txt /usr/share/wordlists/rockyou.txt

-m specifies the hash-type given by hashcat wiki.

-T speciefies the number of core used for the execution.

-a pecifies the attack type:

  • 0 wordlist attack
  • 1 combined attack
  • 3 mask attack (brute-force)
  • 6 hybrid attack (wordlist+mask)

-o specifies the filename the cracked hash is to be stored followed by the file name that contains the hash — here `HASH.txt` and finally the path to the wordlist.


Useful options

-w: Sets workload profile for devices (1 = slow and low power consumption; 4 = fast, high performance).

-p: Defines a separator for hashes in a hash file.

--force: Forces Hashcat to ignore some errors or warnings.

--status: Displays the current status of the attack.


Managing GPU/CPU Devices:

-D 1: Uses CPU only.

-D 2: Uses GPU only.

-I: Identify available devices.


Benchmark & more informations

To test your machine’s performance across different hash types:

hashcat -b
