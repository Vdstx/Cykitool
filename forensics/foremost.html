Foremost

About Foremost

Foremost is a forensic data recovery tool used to recover files based on their headers, footers, and internal data structures. It was originally developed by the United States Air Force and is now widely used by cybersecurity professionals for file carving during digital forensics investigations. Foremost works on disk images, memory dumps, or any raw data to extract files of various types, such as images, documents, executables, and more.


Installation

Linux installation

sudo apt install foremost

Alternatively, you can build from source:

git clone https://github.com/kfairbanks/foremost
cd foremost
make
sudo make install

Examples of Foremost Usage

Basic file recovery:

foremost -i disk_image.dd -o /recovery_directory/

  • -i: Specifies the input file (a disk image in this case).
  • -o: Specifies the output directory where recovered files will be saved.

Recovering specific file types (e.g., JPEG images):

foremost -t jpg -i disk_image.dd -o /recovery_directory/

  • -t: Specifies the file type to recover (e.g., jpg for JPEG files).
  • -i: Specifies the input file (disk image).
  • -o: Specifies the output directory for recovered files.

Running Foremost with a configuration file:

foremost -c foremost.conf -i disk_image.dd -o /recovery_directory/

  • -c: Specifies a custom configuration file to define file carving parameters.

Useful Options

  • -v: Enables verbose mode for detailed output during file recovery.
  • -Q: Quick mode, which skips header/footer integrity checks for faster recovery.
  • -t: Specifies the file types to carve (e.g., jpg, pdf, exe).
  • -c: Use a custom configuration file.
  • -s: Skip a specified number of bytes at the beginning of the input file.
  • -T: Enables thorough mode to attempt recovery of fragmented files.
