Aircrack-ng

About Aircrack-ng

Aircrack-ng is a comprehensive suite of tools designed for assessing Wi-Fi network security. It focuses on monitoring, attacking, testing, and cracking Wi-Fi networks. Aircrack-ng is widely used for network audits and penetration testing to evaluate the security of wireless encryption protocols like WEP, WPA, and WPA2.


Installation

On Debian-based Linux distributions:

sudo apt-get install aircrack-ng

On macOS using Homebrew:

brew install aircrack-ng

To build from source:

git clone https://github.com/aircrack-ng/aircrack-ng
cd aircrack-ng
./autogen.sh
./configure
make
sudo make install

Examples of Aircrack-ng Usage

Putting Wireless Interface into Monitor Mode:

sudo airmon-ng start wlan0

  • wlan0: Your wireless network interface.

Capturing Packets with Airodump-ng:

sudo airodump-ng wlan0mon

  • wlan0mon: The wireless interface in monitor mode.

Filtering by Channel and BSSID:

sudo airodump-ng --bssid AA:BB:CC:DD:EE:FF --channel 6 --write capture wlan0mon

  • --bssid: Filters packets from a specific access point.
  • --channel: Specifies the Wi-Fi channel to monitor.
  • --write: Writes the captured packets to a file.

Deauthenticating Clients with Aireplay-ng:

sudo aireplay-ng --deauth 10 -a AA:BB:CC:DD:EE:FF wlan0mon

  • --deauth 10: Sends 10 deauthentication frames.
  • -a: Specifies the BSSID of the target access point.

Cracking WEP Encryption:

aircrack-ng -b AA:BB:CC:DD:EE:FF capture*.cap

  • -b: Specifies the BSSID of the target network.
  • capture*.cap: The capture files containing the IVs.

Cracking WPA/WPA2 Encryption:

aircrack-ng -w wordlist.txt -b AA:BB:CC:DD:EE:FF capture.cap

  • -w: Specifies the wordlist file for dictionary attack.
  • -b: Specifies the BSSID of the target network.
  • capture.cap: The capture file containing the handshake.

Useful Options

Airodump-ng Options

  • --ivs: Only capture initialization vectors (IVs).
  • --encrypt: Filter networks by encryption type (WEP, WPA, WPA2).
  • --essid: Filter networks by ESSID.
  • --output-format: Specify output file format (pcap, ivs, csv, kismet, gps).

Aireplay-ng Options

  • --fakeauth: Perform fake authentication with the access point.
  • --arpreplay: Replay ARP requests to generate traffic.
  • -h: Specify the MAC address of the client.

Aircrack-ng Options

  • -a: Force attack mode (1=WEP, 2=WPA/WPA2).
  • -e: Target network ESSID.
  • -l: Write the found key to a file.
  • -q: Quiet mode, less verbose output.
  • -K: Use PTW attack against WEP (faster cracking).

Additional Tools in Aircrack-ng Suite

  • Airmon-ng: Enables monitor mode on wireless interfaces.
  • Airodump-ng: For packet capturing of raw 802.11 frames.
  • Aireplay-ng: Packet injection tool to generate traffic.
  • Airbase-ng: Used for configuring rogue access points.
  • Airdecap-ng: Decrypts WEP/WPA/WPA2 capture files.
  • Packetforge-ng: Creates custom packets for injection.
  • Airtun-ng: Virtual tunnel interface creator.
  • Airgraph-ng: Graphing tool to visualize networks.

Troubleshooting and Tips

  • Monitor Mode: Use airmon-ng start wlan0 to enable monitor mode on your wireless interface.
  • Interference: Kill interfering processes with airmon-ng check kill to prevent network managers from disrupting your operations.
  • Root Privileges: Most Aircrack-ng tools require root access; use sudo before commands.
  • Wireless Adapter: Ensure your wireless adapter supports monitor mode and packet injection.
  • Channel Locking: Lock the channel with --channel to avoid missing packets due to channel hopping.
  • Wordlists: Use comprehensive wordlists for WPA/WPA2 cracking, like those from SecLists.

Legal Considerations

Always obtain proper authorization before using Aircrack-ng to test network security. Unauthorized access to networks is illegal and unethical.
