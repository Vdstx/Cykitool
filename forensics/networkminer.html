NetworkMiner

About NetworkMiner

NetworkMiner is an open-source network forensic analysis tool (NFAT) for Windows. It can be used as a passive network sniffer/packet capturing tool to detect operating systems, sessions, hostnames, open ports, and more without putting any traffic on the network. NetworkMiner can also parse PCAP files for offline analysis and extract files and certificates transferred over the network.


Installation

NetworkMiner is available for Windows, but it can run on Linux and macOS using Mono.

  • Download link: Official NetworkMiner Download

On Windows:

  1. Download the ZIP file from the official website.
  2. Extract the contents to a desired location.
  3. Run NetworkMiner.exe as Administrator.

On Linux/macOS using Mono:

  1. Install Mono:
    2. sudo apt-get install mono-complete
  2. Download and extract NetworkMiner.
  3. Run NetworkMiner using Mono:
    4. mono NetworkMiner.exe

Examples of NetworkMiner Usage

Live Network Sniffing:

  1. Launch NetworkMiner with administrative privileges.
  2. Select your network interface from the dropdown menu.
  3. Click on Start to begin capturing network traffic.
  4. NetworkMiner will passively collect data and display hosts, sessions, and extracted files.

Analyzing PCAP Files:

  1. Open NetworkMiner.
  2. Go to File > Open PCAP.
  3. Select the PCAP file you wish to analyze.
  4. NetworkMiner will parse the file and display the extracted information.

Extracting Transferred Files:

  1. After capturing or loading a PCAP file, navigate to the Files tab.
  2. NetworkMiner will list files transferred over the network, categorized by protocol.
  3. Right-click on a file to save or open it.

Useful Features

  • Host Identification: Provides details about each host, including IP addresses, MAC addresses, OS, and open ports.
  • Credential Extraction: Extracts credentials such as usernames and passwords transmitted over the network (when not encrypted).
  • File Carving: Automatically extracts files from network traffic, including images, documents, and executables.
  • DNS and Protocol Analysis: Displays DNS queries and supports various protocols like HTTP, FTP, SMTP, SMB, and more.
  • SSL/TLS Certificates: Extracts and displays SSL/TLS certificates exchanged during secure sessions.
  • Custom Scripts: Supports extensions through custom scripts for advanced analysis.

Troubleshooting and Tips

  • Promiscuous Mode: Ensure your network interface supports promiscuous mode to capture all network traffic.
  • Administrative Rights: Run NetworkMiner with administrative privileges to access network interfaces properly.
  • Passive Sniffing: NetworkMiner does not transmit packets; use it for passive monitoring to avoid network disruptions.
  • Encryption Limitations: Encrypted traffic (e.g., HTTPS) cannot be decrypted without the appropriate keys.
  • Use with Wireshark: Combine NetworkMiner with Wireshark for comprehensive network analysis.
  • Filtering: Use filters to focus on specific hosts, protocols, or types of data.

Example: Extracting Images from Network Traffic

If you want to extract images transferred over HTTP from a PCAP file:

  1. Open NetworkMiner and load the PCAP file.
  2. Navigate to the Images tab.
  3. All images detected in the network traffic will be displayed.
  4. You can preview and save the images for further analysis.

Legal Considerations

Ensure you have the legal right and authorization to capture and analyze network traffic. Unauthorized interception of network communications is illegal and unethical.
